## 第一条 服务范围

### 1.1 托管基础版

麟创为客户提供以下服务：

1. **平台部署**：远程一键部署 VigilOps 监控平台（含 Backend、Frontend、PostgreSQL、Redis 四个容器）
2. **主机监控**：含 1 台主机的 Agent 部署与指标采集（CPU/内存/磁盘/网络/负载）
3. **服务监控**：服务发现、健康检查、可用率计算
4. **告警体系**：告警规则配置、告警通知（邮件/钉钉/飞书/企微任选）
5. **AI 日志分析**：共享 AI 配额，含日志异常分析、AI 对话、告警根因分析
6. **Dashboard**：实时仪表盘（WebSocket 推送）、健康评分、趋势图表
7. **日志管理**：日志采集、搜索、实时流
8. **运维报告**：自动日报/周报 + 月度人工总结
9. **巡检服务**：每周 AI 自动巡检
10. **平台升级**：季度版本升级

### 1.2 托管专业版

在基础版所有服务之上，额外包含：

1. **自动修复系统**：6 个内置 Runbook（磁盘清理、服务重启、僵尸进程清理、内存释放、日志轮转、连接重置）+ 审批流
2. **数据库监控**：含 3 个数据库实例（PostgreSQL/MySQL/Oracle），含连接数、慢查询、QPS 等指标
3. **SLA 管理**：SLA 规则配置 + 违规检测 + 月度 SLA 达成报告
4. **服务拓扑图**：可视化服务依赖关系 + AI 拓扑建议
5. **独立 AI 配额**：优先处理 + 运维记忆系统（AI 记住历史故障处理经验）
6. **巡检服务**：每日 AI 巡检 + 人工复核
7. **平台升级**：月度版本升级 + hotfix
8. **运维报告**：日/周/月报 + 定制分析

## 第三条 费用及支付方式

### 3.1 服务费用

具体费用以双方确认的《报价单》为准，主要包括：

- 平台实例费（按月计费）
- 额外主机监控费（按台/月计费）
- 额外数据库监控费（按个/月计费）

### 3.2 支付方式

1. 按月支付，每月 [1] 日前支付当月费用
2. 支持银行转账、支付宝对公转账
3. 麟创在收到款项后 3 个工作日内开具发票

### 3.3 费用调整

服务费用如需调整，麟创应提前 30 天书面通知客户。客户有权在通知期内决定是否续约。

## 第四条 甲方责任

1. 按照本协议约定提供运维托管服务
2. 保证监控平台正常运行，定期巡检和维护
3. 及时响应客户报告的故障和问题
4. 定期提供运维报告
5. 对客户数据和系统信息严格保密
6. 平台版本升级前提前通知客户并协商维护窗口

---

## 第五条 乙方责任

1. 按时支付服务费用
2. 提供必要的服务器访问权限（SSH 密钥/密码）
3. 配合麟创完成平台部署和 Agent 安装
4. 及时通知麟创任何基础设施变更（如服务器迁移、网络调整）
5. 不对 VigilOps 平台进行未经授权的修改
6. 确保被监控服务器的基本网络连通性

## 第六条 保密条款

1. 双方在合作过程中知悉的对方商业秘密、技术信息、客户数据等均属保密信息
2. 未经对方书面同意，任何一方不得向第三方披露保密信息
3. 保密义务在本协议终止后 2 年内继续有效
4. 以下情形除外：
   - 已公开的信息
   - 非因违约方原因而公开的信息
   - 法律法规要求披露的信息

---

## 第七条 数据安全

1. 麟创仅采集客户授权范围内的监控数据（系统指标、日志、服务状态）
2. 采集的数据存储在客户自有服务器上，麟创不擅自外传
3. AI 分析调用外部 API 时，仅传输必要的脱敏数据
4. 客户有权随时要求导出或删除其数据

## 第八条 协议期限与终止

### 8.1 协议期限

本协议自签订之日起生效，有效期为 [12] 个月，期满后自动续期，每次续期 [12] 个月。

### 8.2 终止条件

1. 任何一方可提前 30 天书面通知对方终止本协议
2. 一方严重违约且经书面催告后 15 天内未纠正的，另一方可立即终止本协议
3. 客户连续 2 个月未支付服务费用的，麟创有权暂停服务并终止本协议

### 8.3 终止后处理

1. 麟创协助客户完成数据导出和交接
2. VigilOps 为开源软件，客户可继续自行使用，但不再享受托管服务
3. 已支付的费用不予退还，未到期部分按实际天数折算退还

## 第九条 免责条款

以下情形麟创不承担责任：

1. 因不可抗力（自然灾害、战争、政策变化等）导致的服务中断
2. 因客户自行操作失误导致的系统故障
3. 因第三方服务（如 AI API 提供商、云服务商）故障导致的功能不可用
4. 因客户基础设施（网络、硬件）问题导致的监控中断
