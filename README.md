# VigilOps

🛡️ 开源 AI 智能运维监控平台

**服务器监控 · 服务监控 · 日志管理 · 数据库监控 · LLM API 管理 · AI 智能分析**

一个 `docker compose up` 搞定。

## 功能

- 📊 多服务器监控（CPU/内存/磁盘/网络）
- 🔍 应用服务健康检查（HTTP/TCP 拨测）
- 📝 日志采集、归档、搜索
- 🧠 AI 智能日志分析（异常检测/根因定位/修复建议）
- 🗄️ 数据库监控（连接数/慢查询/QPS）
- 🤖 LLM API 代理网关 + 费用管理
- 🔔 统一告警中心 + AI 告警降噪
- 📱 多渠道通知（企业微信/飞书/钉钉/邮件/短信）

## 技术栈

- **后端**: Python + FastAPI
- **前端**: React + TypeScript + Ant Design + ECharts
- **Agent**: Python（轻量采集端）
- **数据库**: PostgreSQL + Redis
- **AI**: 支持 DeepSeek / OpenAI / Anthropic / Ollama
- **部署**: Docker Compose
